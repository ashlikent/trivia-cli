# Trivia CLI Game

## Description
A command-line trivia game built with Node.js that asks users questions, tracks score, and displays results with a timer.

## Features
- Interactive CLI prompts
- Real-time answer feedback
- Score tracking
- Game timer

## How to Run
1. Install dependencies:
   npm install

2. Run the game:
   node index.js

## Technologies Used
- JavaScript
- Node.js
- Inquirer
